The simplest possible plugin to compute the days to a specified date.

== Installation ==

1. Upload the `days-to` plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode where you want to insert the days to a date.

Example: [days-to date='2023-07-20T08:00:00-05']

== Frequently Asked Questions ==

= What happens when the date is in the past? =

The number of days returned is negative.

= Why doesn't days-to do *X*? =

Feature requests can be entered at the [days-to repo on GitHub](https://github.com/voidampersand/days-to).
